## INSTALLATION

```
docker run -d --name dropbox -v ${HOME}:/home/dropbox --restart=always alexandreoda/dropbox
```


## CONFIGURATION

Utilisez les logs du container au premier lancement pour obtenir le lien de synchronisation dropbox

```
docker logs dropbox
```
